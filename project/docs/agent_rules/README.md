# AI運用ルール一覧

このフォルダは、AntigravityとCodex CLIが必要に応じて読む詳細ルールを置く場所です。

## ファイル一覧

- `github_workflow.md`
  - Git、commit、push、PR、GitHub整理の手順。
- `codex_collaboration.md`
  - Codex CLIの使い方、AntigravityからCodexへ渡す依頼文の作り方。
- `search_research_workflow.md`
  - Context7、Tavily、Google Developer Knowledge、GitHub検索などの使い分け。
- `paper_to_code_workflow.md`
  - 情報系論文を実装仕様書へ変換する手順。
- `external_code_policy.md`
  - 論文公式GitHub、外部リポジトリ、データセットの扱い。
- `error_investigation.md`
  - エラー原因調査をCodexへ委任する手順。

## 基本方針

- 常時読むルールは短くする。
- 詳細な説明はこのフォルダに分ける。
- 調査結果は会話に残すだけでなく、Markdownファイルとして保存する。
- 実装は、調査メモと実装計画を読んでから最小単位で進める。
